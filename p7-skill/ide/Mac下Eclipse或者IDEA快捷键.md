# IDEA快捷键目录
官方文档上的位置是:

```
~/Library/Application Support/JetBrains/<product><version>/keymaps
```

Win:

```
%APPDATA%\JetBrains\<product><version>\keymaps
```

# IDEA快捷键
| 快捷键名称 | IDEA            | Eclipse         |
|:------|-----------------|-----------------|
| 快速搜索  | `Ctrl+H`        | `Ctrl+H`        |
| 前面复制行 | /               | `CMD+Alt+Up`    |
| 下面复制行 | `CMD+Alt+Down`  | `CMD+Alt+Down`  |
| tab切换 | `CMD+Alt+Left`  | `CMD+Alt+Left`  |
| tab切换 | `CMD+Alt+Right` | `CMD+Alt+Right` |
| 查找    | `CMD+F`         | `CMD+F`         |
| 替换    | `CMD+R`         | /               |
